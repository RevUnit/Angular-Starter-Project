## Setup
-----------------------------------------------------------
```sh
$ nvm use
$ npm install
$ npm run start
```

## NPM Commands
-----------------------------------------------------------
```sh
$ npm run lint // Runs SASS linter and code linter
$ npm run test // Runs tests
$ npm run build // Builds project for deployment
```
Note: Application builds are stored in the `dist/` directory
